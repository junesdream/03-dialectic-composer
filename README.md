# 🎹 DIALECTIC_COMPOSER v1.2

An algorithmic music synthesizer built with **React**, **TypeScript**, and **Tone.js**. This application translates philosophical concepts into unique harmonic structures using real-time digital signal processing.

## 🚀 Concept: Musical Synthesis
Based on the Hegelian dialectic, the app takes two opposing inputs:
1. **Thesis:** Generates the harmonic foundation.
2. **Antithesis:** Introduces rhythmic and melodic friction.
3. **Synthesis:** A mathematical fusion of both frequency vectors, resulting in a unique generative composition.

## 🛠️ Technical Features
* **Web Audio Engine:** Powered by `Tone.js` for low-latency synthesis and reverb effects.
* **Real-time Oscilloscope:** Custom `<canvas>` visualizer mapping the raw waveform data.
* **Mathematical Mapping:** Concepts are converted to frequencies via ASCII-sum algorithms, ensuring every word has a unique "sonic fingerprint".
* **Cyberpunk UI:** High-contrast Matrix-inspired design with a "Silver Neon" aesthetic.
* **Keyboard Workflow:** Fully operational via `Enter` key for a seamless "Terminal" experience.
